# 🛒 E-Commerce Sales Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on an e-commerce sales dataset using Python. The objective is to understand sales patterns, product performance, customer reviews, and category-wise trends while creating a new feature (**Total Sales**) for analysis and prediction.

The project includes data cleaning, preprocessing, feature engineering, visualization, and encoding techniques.

---

## 🎯 Objective

- Analyze the e-commerce sales dataset.
- Check and clean missing values.
- Create a new feature called **Total Sales**.
- Explore sales trends using visualizations.
- Encode categorical variables for machine learning.
- Prepare the dataset for predictive modeling.

---

## 📂 Dataset Information

The dataset contains information such as:

- Product ID
- Product Name
- Category
- Price
- Quantity Sold
- Review Score
- Total Sales (Created during the project)
